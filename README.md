# SSH Brute Force Tool

This tool is designed to demonstrate SSH brute force attacks. Use it responsibly and only on systems you have explicit permission to test.

## Usage

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the script:
   ```
   python ssh_brute_force.py <target_ip> <username> <wordlist.txt>
   ```

This tool is for educational purposes only. Unauthorized use is illegal.
